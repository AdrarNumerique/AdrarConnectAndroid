WebServices

Decription détaillée des web services utilisés par les applications mobiles Adrar Connect.

Ces webservices gèreront les données

  • des candidats,
  • les données statiques envoyées depuis l'index,,
  • des sessions d'informations collectives,
  • des formations
  • de la FAQ,
  • des documents,
  • des types des document,
  • des erreurs

Détails des web services

Signin

POST /ws/signin
Renvoie le candidat comprenant un idSessionConnexion
Renvoie 'Err_Email_Already_Exist' si l'email existe déja en bdd
Entrée

Login

POST /ws/login
Renvoie le candidat comprenant un idSessionConnexion
Renvoie 'Err_Mdp' si le mdp est erroné
Renvoie 'Err_Email' si le l'email n'existe pas en bdd
Entrée

Formulaire informations candidat

POST /ws/candidat/update/details/
Renvoie le candidat mis à jour
Entrée

Envoi document

POST /ws/candidat/update/doc/
Renvoie le candidat mis à jour
Entrée

Inscription infos collectives

POST /ws/infos-collectives/inscription
Renvoie le candidat mis à jour
Entrée

Changement inscription infos collectives

POST /ws/infos-collectives/update/
Renvoie le candidat mis à jour
Entrée

Annuler inscription infos collectives

GET /ws/infos-collectives/delete/
Renvoie le candidat mis à jour

Données statiques

GET /ws/accueil
Lors de la connexion à l'index, le serveur envoie un objet de données statiques

Erreurs

GET /ws/erreur
Renvoie une erreur
